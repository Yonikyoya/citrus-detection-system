# 用户管理系统操作手册

## 1. 用户注册
1. 访问 `/register` 页面。
2. 输入用户名。
3. 输入密码及确认密码。
4. 点击“注册”。注册成功后，系统将为您生成唯一的 UserID。

## 2. 用户登录
1. 访问 `/login` 页面。
2. 输入用户名或邮箱，以及密码。
3. 输入右侧显示的算术验证码答案。
4. 勾选“记住我”可延长登录会话。
5. 点击“登录”。

## 3. 个人中心
- **更新资料**: 登录后可修改昵称、头像和联系电话。
- **修改密码**: 修改密码属于敏感操作，需验证旧密码，并输入发送至邮箱的二次验证码。

## 4. 管理员功能 (仅 admin 角色)
- **用户列表**: 访问 `/api/admin/users` (通过后台管理界面调用)，支持搜索、筛选和分页。
- **禁用/启用账户**: 可一键冻结违规账户，冻结后用户将无法登录。
- **权限分配**: 可修改其他用户的角色（如将 Worker 提升为 Owner 或 Admin）。
- **操作审计**: 管理员的所有关键操作（如修改他人信息、冻结账户）均会被记录在审计日志中，包含 IP 和时间戳。

## 5. 安全说明
- **暴力破解防护**: 连续登录失败 5 次将锁定账户 30 分钟。
- **密码存储**: 使用 bcrypt 强哈希加密，无法被逆向破解。
- **数据隔离**: 每个用户仅能访问自己关联果园的检测数据。
